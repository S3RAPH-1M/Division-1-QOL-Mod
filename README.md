# Division-1-QOL-Mod

A simple Quality of Life mod for **Tom Clancyâ€™s The Division (2016)** that adds useful in-game customization features focused on visuals and atmosphere.

## Features

### Field of View (FOV) Changer
- Adjust the player camera Field of View
- Increase or decrease FOV to your preference
- Includes an optional **Aim Down Sights (ADS) zoom** feature

### Time of Day Changer
- Change the current in-game time
- Select a specific time of day
- Includes a button to smoothly transition to the selected time

### Weather Changer
- Modify the current weather environment
- Two weather modes:
  - **Current Weather**
  - **Blend Weather**
- Includes a time option to control how the weather transition occurs

## Purpose

This mod is designed to improve visual comfort and immersion without altering gameplay balance. All features are optional and configurable.
